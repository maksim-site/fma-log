# FMA Logistics website concept

Static one-page website concept for FMA Logistics.

## Files

- `index.html` - recommended light corporate version for GitHub Pages.
- `v2.html` - same light version, kept as the original concept file.
- `concept-dark.html` - earlier dark visual concept.
- `logo.png` - FMA Logistics logo used in the layout.

## Preview

Open `index.html` in a browser, or publish this folder with GitHub Pages.

For GitHub Pages:

1. Create a new repository, for example `fma-log`.
2. Upload these files to the repository root.
3. Open `Settings -> Pages`.
4. Select `Deploy from a branch`, branch `main`, folder `/root`.
5. Send the generated Pages URL to the client.
